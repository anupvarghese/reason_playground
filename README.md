# Set up OCaml & Opam

```shell
brew install ocaml
brew install opam
opam upgrade
opam install reason
opam install merlin
```

# Basic Reason Template

Hello! This project allows you to quickly get started with Reason and BuckleScript.

`npm install -g bs-platform`
`bsb -init reason-playground -theme basic-reason`

# Build
```
npm run build
```

# Build + Watch

```
npm run watch
```
